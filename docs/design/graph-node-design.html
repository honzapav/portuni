A small disc
that says everything.

A focused pass at Constellation pip — direction D from the studies. The premise: keep the disc small, encode lifecycle through the disc's own fill pattern, let the rest of the canvas stay quiet. Below: a full pattern dictionary, the lifecycle map for every entity type, three placements for the owner, and a zoom-level survival check.

03

Where the owner lives — three options.

The original D parked the owner in a chip beneath the label, which adds a third vertical line per node. Two leaner alternatives below — and one that earns its keep.

D-1 · caption below

vertical sprawl

Three lines under every node. Easy to read, but the canvas grows tall fast.

D-2 · inline

balanced

Owner blends into the label as a monospaced suffix. No extra line, but only readable up close.

D-3 · pip on disc

★ Recommended

A 9-px notch tucks into the disc at 4 o'clock. Owner becomes a permanent fixture of the node, not a dependent caption — survives zoom-out far better than D-1 or D-2.

04

Zoom-level survival — three altitudes.

The graph is panned and zoomed constantly. Patterns must remain decipherable at the zoomed-out flyover, and the owner pip must not become noise.

Far · 35 %

Pattern is the only signal that survives. Owner pip disappears; node shapes still tell the story (struck, hatched, half, solid, ghost).

Medium · 100 %

Pip is readable; pattern + owner cohabit comfortably. The default working zoom.

Close · 200 %

Pattern, label, owner — all crisp. The detail pane is one click away; this is decision-making zoom.

05

Shipping it — what changes in code.

Most patterns are achievable with Cytoscape's existing styling primitives, plus a small data-URI generator for the four overlay patterns. No new layout engine.

Pattern delivery

Cytoscape's background-image on a node accepts a data URI. Build a tiny helper that, given (type, lifecycle_state), returns:

  • solid, ghost — no overlay; existing background-color + opacity
  • hollow, dashed-hollow, dotted-hollow — set background-opacity: 0 and use border-style + border-width
  • checked, struck — small inline SVG, layered as background-image
  • hatched, stippled, half-fill, three-quarter — repeating SVG pattern data URI

Owner pip (D-3)

Cytoscape doesn't render satellite badges natively. Two paths:

  • HTML overlay — render owner pips as absolutely-positioned divs inside the Cytoscape container, repositioned on each render event. Cheapest visually.
  • Sibling node — add invisible compound parent + child "pip" node per entity, positioned via constraint. Heavier but stays inside Cytoscape's coordinate system.

Ship HTML overlay first; revisit if pip count exceeds ~150 nodes per viewport.
